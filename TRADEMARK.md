# QUHAIM Trademark Policy

The source code for QUHAIM Toolkit Pro is open source, but the project identity is protected.

## Protected Marks

The following names and assets are trademarks or protected project identity of QUHAIM Labs:

- QUHAIM
- QUHAIM Labs
- QUHAIM Toolkit Pro
- QUHAIMToolkitPro
- Official icons, logos, setup branding, and release artwork

## Allowed Use

You may use the protected marks to truthfully refer to the original project, for example:

- "Forked from QUHAIM Toolkit Pro"
- "Compatible with QUHAIM Toolkit Pro plugin structure"
- "Based on QUHAIM Toolkit Pro source code"

## Not Allowed Without Permission

You may not:

- Publish a modified build as "QUHAIM Toolkit Pro".
- Use QUHAIM icons or logos for unofficial releases.
- Present a fork as an official QUHAIM Labs product.
- Register confusingly similar names, domains, packages, or repositories.

## Fork Requirements

If you distribute a modified build, rename the product and remove QUHAIM branding from the application name, installer name, icons, and release pages.

Official releases are distributed only by QUHAIM Labs or approved maintainers.
