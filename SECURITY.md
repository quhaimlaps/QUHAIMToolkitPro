# Security Policy

## Supported Versions

Security review currently applies to the active QUHAIM Toolkit Pro release line.

## Reporting a Vulnerability

Do not publish sensitive vulnerability details publicly before maintainers have time to review them.

Report issues through the official repository issue tracker or a private maintainer channel when available.

Include:

- A clear description of the issue.
- Steps to reproduce.
- Affected files or plugins.
- Whether administrator privileges are required.

## Installer Integrity

Official installers should be downloaded from official QUHAIM Labs release channels. Future production builds should be code-signed.
